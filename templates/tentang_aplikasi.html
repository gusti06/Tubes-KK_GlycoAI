Tentang Aplikasi

GlycoAI untuk project final mahasiswa

Aplikasi ini dibuat dengan Flask di sisi backend, Bootstrap 5 untuk UI, dan implementasi Fuzzy Mamdani yang sederhana namun nyata. Struktur folder dipisah agar mudah dipahami saat presentasi maupun pengembangan lanjut.

Alur teknis

  1. Pengguna mengisi form prediksi.
  2. Frontend mengirim data ke API Flask.
  3. Backend menjalankan fuzzification, inference, dan defuzzification.
  4. Hasil ditampilkan bersama chart dan rekomendasi.

Komponen utama

  • Landing page modern dan responsif
  • Halaman penjelasan diabetes
  • Halaman tentang aplikasi
  • Dashboard statistik hasil prediksi
  • Loading animation dan validasi form

Catatan implementasi

Riwayat prediksi disimpan lokal agar total prediksi pengguna dan grafik statistik tetap terlihat hidup saat demo. Dataset sample Pima Indians juga disediakan di folder dataset untuk menunjukkan basis data yang digunakan.
